1.object 在程式語言中物件通常以{ }表示，個人認為他有點像 array(陣列)都是資料的集合。

2.而物件的特色在它擁有屬性與值(key&value)，給予特定的 key 可以取得對應的 value。

舉個例子，我們常使用電子信箱。然而一個網站會有很多的用戶，因此當你要使用時你必須要告訴網誌你的郵箱地址或是帳號並給予他密碼你才能夠登入。

3.新增物件與刪除物件：利用物件名稱.屬性＝值去新增物件，用 delete 指令刪除物件。

例如：原本家裡有四個人爺爺、爸爸、媽媽和我今天爺爺過世了用 delete 指令爺爺就被刪除了。

隔沒多久妹妹出生了，利用物件名稱.屬性＝值妹妹就在家中出現了。

碎碎念：

1.在 Js 中只有兩種分類純值和物件。

2.物件實字 (Object Literals)：const parentHouse = { }這種建立物件的方式叫做物件實字

3.建構式(Constructor)：const myHouse = new Object(parentHouse) 用來建立新的物件。
